This assignment is designed to help students strengthen their understanding of HTML fundamentals by creating multiple web pages using different HTML elements and formatting tags.
Students will practice building properly structured HTML documents, working with headings, paragraphs, comments, quotations, text formatting, and preformatted content.

The assignment also encourages clean coding practices, proper organization, and creativity while helping learners gain confidence in writing HTML manually.
