# HTML Fundamentals Assignment

This repository contains a set of HTML practice files created to build foundational web development skills. Each task focuses on a core HTML concept, helping me learn how to write structured, readable, and semantic HTML documents.

## Files Included

- `task1-headings.html` - Practice using heading tags from `<h1>` to `<h6>`.
- `task2-biography.html` - Create a simple biography page using paragraphs, strong text, underlines, lists, and italics.
- `task3-quotes.html` - Display quotations using `<blockquote>` and `<cite>`.
- `Task4-text-formatting.html` - Use text-formatting tags such as `<del>`, `<ins>`, `<small>`, and `<mark>`.
- `task5-structuredhtml.html` - Create a structured HTML page using `<section>`, headings, and paragraphs.
- `task6-preformattedtext.html` - Work with preformatted text using the `<pre>` element.
- `task7-combinedformatting.html` - Combine formatting tags to reinforce HTML structure and styling.
- `task8-comments.html` - Demonstrate the use of HTML comments (`<!-- ... -->`) in a document.
- `task9-quotesDetail.html` - Add more quotes and citation details with `<blockquote>` and `<cite>`.

## What I Have Learned from These Projects

These tasks have helped me develop the following HTML skills:

- Writing valid HTML documents with `<!DOCTYPE html>` and proper `<html>`, `<head>`, and `<body>` structure.
- Using heading tags `<h1>` through `<h6>` to organize content hierarchically.
- Adding metadata with `<meta charset="UTF-8">` and responsive settings using `<meta name="viewport">`.
- Creating paragraphs with `<p>` and using line breaks with `<br>`.
- Emphasizing text using `<strong>`, `<b>`, `<em>`, `<i>`, `<u>`, `<small>`, and `<mark>`.
- Inserting editorial changes and review-style formatting with `<del>` and `<ins>`.
- Building lists using `<ul>` and `<li>` for structured content.
- Using semantic grouping with `<section>` to separate page content.
- Adding quotes and attribution using `<blockquote>` and `<cite>`.
- Writing preformatted content with `<pre>` to preserve whitespace and layout.
- Including comments in HTML to document code and keep the file readable.
- Practicing clean markup and basic page composition.

## Summary

This assignment collection is a strong first step in HTML fundamentals. It helps reinforce the basics of page structure, text formatting, quotations, content organization, and comments.
